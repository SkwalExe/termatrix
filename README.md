# Installation

First, clone this repository

```bash
git clone https://github.com/SkwalExe/termatrix
cd termatrix
```

Then, install the dependencies

```bash
pip3 install -r requirements.txt
```

And finally, launch the program and follow the instructions

```bash
python3 src/main.py
```
